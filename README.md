# 💫 About Me

I'm **Rahim Kamran**, an MCA student at **Lovely Professional University** with a BCA background from the same institution. I'm a passionate **Data Science and Machine Learning enthusiast** with hands-on experience in Python, SQL, Power BI, Java, C++, and full-stack web development.

I completed a **Cloud Backend Internship at VVDN Technologies**, where I built secure cloud recording systems and RESTful APIs for a large-scale surveillance platform. I've independently developed projects including a **House Price Prediction ML model** and a **full-stack Expense Tracker**, and hold certifications in Oracle Data Science and Machine Learning from Internshala.
