# 💫 About Me:
Rahim Kamran is a results-driven Computer Science postgraduate from Lovely Professional University (LPU), currently pursuing his Master of Computer Applications (MCA) after completing his Bachelor of Computer Applications (BCA) from the same institution with a CGPA of 7.5.<br>With a strong foundation in data science, machine learning, and full-stack development, Rahim brings both analytical depth and practical engineering experience to every project he undertakes. His technical skill set spans Python, Java, C++, SQL, and web technologies including HTML, CSS, JavaScript, PHP, and React — complemented by hands-on expertise in machine learning libraries such as Scikit-learn, Pandas, NumPy, and Matplotlib.<br>Rahim's professional journey includes a Cloud Backend Internship at VVDN Technologies Pvt. Ltd., where he contributed to the development of a unified CP PLUS surveillance platform — architecting secure cloud-based recording systems, integrating RESTful APIs, and optimizing backend performance for high-volume CCTV data handling across devices.<br>On the project front, he has independently built a House Price Prediction model using regression algorithms evaluated against real-world metrics, and developed a full-stack Expense Tracker Web Application complete with secure authentication, CRUD functionality, and dynamic financial dashboards — both available on his GitHub.<br>Rahim holds certifications in Oracle Data Science and Complete Machine Learning (Internshala), reflecting his commitment to continuous learning in the field of data and AI.<br>Currently based in Rampur, Uttar Pradesh, he is open to opportunities in data science, machine learning engineering, and backend development — driven by a passion for building systems that turn data into decisions.
